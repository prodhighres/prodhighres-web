Angeschaut ___________________________________________________________

Node.js - ist ein Programm das es erlaubt JavaScript direkt auf dem Computer lokal anstatt im Browser auszuführen.
-> wird gebraucht, weil viele Tools wie Vite, React etc auf JavaScript basieren

npm - ist der Node Package Manager von Node.js. Damit installiert man Libraries wie react oder supabase
- eine library ist eine Sammlung von code, der importiert werden kann. React zum Beispiel ermöglicht grosse Änderungen am Aussehen mit wenig Javascript code zu schreiben und baut dann den echten "übersetzten" javascript code.
- es wird auch gebraucht um einen entwicklungsserver zu starten also einen local host zu erstellen

Git - ist ein Programm, das Versionen des Codes in Branches speichert. Anstatt Versionen immer wieder neu manuell abzuspeichern, kann ein Branch eine Abweichung vom originalcode commited werden, der entweder integriert oder abgelehnt werden kann.
-> braucht MinTTY als UI Ersatz für die Windows Powershell Console. Bessere Font, Copy paste etc.
-> braucht GitBash als Konsole

GitHub 
- ist die Cloudschnittstelle zu Git, es macht das selbe wie Git, ermöglicht aber das Arbeiten mit anderen Menschen, oder mit mehreren Geräten per Internet. Ausserdem werden hier Dinge deployed, das bedeutet an d

- Repository: Dies ist das Projekt
- Branches: Abweichungen vom Originalcode
- Commits: so werden Abweichungen in den Originalcode integriert
